FAQ

"Why is it saying a source directory doesnt contain a CMakeLists.txt file?"

This means that the subdirectories weren't pulled with the repo.
Run git submodule update --init --recursive

"How do I update Lime3DS on 'insert platform here'?"

Currently, there is no automatic updating system for Lime3DS (one is in the works!). You need to manually download the latest release.

"Where can I get games/keys?"

You can dump games from your own 3DS console using the wiki articles. We do not condone piracy and any further mention of it will be a 1 day timeout in discord. Further asking will result in a ban.
